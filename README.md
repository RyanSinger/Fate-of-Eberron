# Fate of Eberron

A Fate adaptation for the Eberron campaign setting.

## What's Here

This repository contains the following documents for playing Fate in Eberron:

- **Eberron Setting Guide** (`setting-guide.md`): world lore, nations, factions, and setting information for GMs and players.

- **Player's Guide** (`players-guide.md`): character creation using archetypes and backgrounds, skills, stunts, dragonmarks, and magic rules.

- **GM's Guide** (`gm-guide.md`): running the game, adversaries, factions as characters, campaign frameworks, and Eberron specific guidance.

- **GM's Toolkit** (`toolkit.md`): exploration rules, vehicle profiles, terrain, delves, and other reference material for play.

- **NPC Gallery** (`npcs.md`): ready to use NPCs organized by type and setting.

- **Play Example** (`play-example.md`): a full scene with 5 iconic PCs demonstrating gameplay mechanics in action.

- **Campaign** (`campaign.md`): campaign specific content, session history, and current state.

## Design Foundations

Fate of Eberron is built on:

- **Fate Condensed** as the base system
- **Archetype and background** character creation (pick an archetype for what you do, a background for where you come from)
- **Atomic Robo** brainstorm and invention mechanics for magic and artifice
- **Fate Adversary Toolkit** principles for NPCs and opposition
